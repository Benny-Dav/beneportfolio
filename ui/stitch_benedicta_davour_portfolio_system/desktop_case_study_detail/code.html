One-Click Portfolio Website Builder

Redefining the onboarding experience for creators by automating complex deployment pipelines into a single, seamless interaction. A study in frontend performance and DX.

Role Lead Frontend & Product
Stack Next.js, Vercel, AWS
Duration 4 Months
Impact +240% Conv. Rate

Core Objectives

  • check_circle Zero-config deployment for static assets
  • check_circle Instant DNS propagation & SSL provisioning
  • check_circle Automated template generation from GitHub

Technical Highlight

"We reduced the time-to-live from 15 minutes to under 45 seconds using edge functions."

The Challenge

The primary barrier for non-technical creators was the "blank canvas" syndrome combined with "deployment fear." Existing tools required manual domain configuration and complex CI/CD knowledge. The goal was to bridge the gap between creative intent and live production without a single line of terminal command.

Onboarding Completion

88% trending_up

A record high for the platform, up from 34% in the previous manual iteration.

speed
45s

Average Deployment Time

Technical Architecture

We leveraged Vercel's Framework API and custom AWS Lambda functions to handle the bulk of the template cloning and environment variable injection. The frontend, built on Next.js 14, utilizes Server Actions to provide immediate UI feedback during the deployment process.

UX Philosophy

The 'Magic Link' approach was applied to everything. No passwords, no credit cards during the first 60 seconds. The faster we could get a user to see their own site on a live URL, the higher the psychological buy-in and retention became.

lightbulb

Key Metric: The 'Aha' Moment

"The moment a user sees their social media handle as 'yourname.ami.com' live in the browser within 30 seconds of landing on the homepage—that's when our conversion funnel stops being a funnel and starts being a flywheel."
